INSTRUCTIONS

1. Launch the game
2. Press START to begin
3. Move the snake with the UP, DOWN, LEFT, and RIGHT keys
4. Eat the apples to make the snake grow bigger
5. Try to eat as many apples as you can!
6. If you run into the walls or yourself, you lose!
7. To pause, press START
8. To restart the game while paused, press SELECT
9. To play again, press START!
